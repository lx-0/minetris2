# MineTris

A mash-up of Minesweeper and Tetris. Tetris pieces fall onto a board seeded with hidden mines — land on one and it's game over. Numbers on revealed cells show adjacent mine counts, just like classic Minesweeper. Clear lines to score points and level up.

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Controls

| Key | Action |
|-----|--------|
| `← →` | Move piece |
| `↑` | Rotate piece |
| `↓` | Soft drop |
| `Space` | Hard drop |

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
